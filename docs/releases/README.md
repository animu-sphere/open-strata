# Release records

Each released version has an immutable record here: its objective, the
capabilities it shipped, compatibility notes, and any known limitations. Release
records are history — once written for a released version they are not rewritten;
new work goes to a new record. Active, incomplete work lives in the
[roadmap](../roadmap/README.md), not here.

| Version | Record | Theme |
| --- | --- | --- |
| v0.12.0 | [v0.12.0.md](v0.12.0.md) | Hosted macOS CI + runtime-contract hardening |

Earlier releases (v0.1.0–v0.11.0) are summarized in the
[roadmap release milestones](../roadmap/README.md#release-milestones); they are
backfilled into dedicated records as the reorganization proceeds.
