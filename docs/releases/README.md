# Release records

Each released version has an immutable record here: its objective, the
capabilities it shipped, compatibility notes, and any known limitations. Release
records are history — once written for a released version they are not rewritten;
new work goes to a new record. Active, incomplete work lives in the
[roadmap](../roadmap/), not here. Granular, dogfooding-derived delivery detail is
retained in the [delivery history](../reports/delivery-history.md).
