# v0.1.0 — first public release

## Objective

Establish the foundation: a VFX Reference Platform aware runtime/build/extension
platform with the static plugin-verification framework, from Phase 0 through the
OpenUSD/MaterialX profiles (Phases 0–3, Phase 4a).

## Shipped capabilities

- Rust workspace + `ost` CLI; machine-readable platform manifests (built-in
  CY2025/CY2026/CY2027 with user overlay); project + lock schemas.
- `ost platform list | show | diff`, `ost init`.
- Runtime + devshell: resolver, profiles (`core`/`dev`/`usd`/`lookdev`), env
  generation, `ost env`/`ost devshell`, `ost runtime pull|list|show` (mock/local),
  digest-bearing `runtime.json`, `ost doctor`, `ost runtime validate`,
  `ost runtime explain`, `strata.lock` via `ost lock`.
- CMake target build: `ost configure|build|package|validate`, Windows MSVC
  bootstrap.
- OpenUSD / MaterialX extension family + capability resolver.
- Static plugin verification (Phase 4a): `ost-plugin`, `ost plugin new|inspect|
  build|doctor` (Levels 0–1), reports + plugin-report schema.
